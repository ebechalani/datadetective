# Jeu d'intro — Data Detectives (30–40 min)

## Matériel
- 1 feuille par groupe (ou cahier)
- La page web `index.html` sert à tirer les cartes (échantillon d'employés)

## Organisation
Groupes de 3–4, rôles :
- Calculateur / Vérificateur / Secrétaire / Porte-parole

## Mission (à faire à la main)
1) Effectifs : combien de F ? combien de M ?
2) Salaire moyen F et salaire moyen M
3) Écart en % : ((moy_H - moy_F) / moy_H) * 100
4) "Poste équivalent" : repérer des binômes même experience + mêmes études, comparer les salaires
5) Conclusion : écrire 2 hypothèses sur l'origine d'un écart

## Lancement
Ouvrir `index.html` (ou GitHub Pages) → cliquer "Nouvelle pioche" → 12 cartes.
